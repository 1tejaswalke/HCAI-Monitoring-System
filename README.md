# 🚗 HCAI Monitoring System

## 📌 Overview

The HCAI Monitoring System is a Human-Centered AI-based solution for real-time driver state monitoring using facial behavior analysis. It detects fatigue, drowsiness, and attention levels to enhance safety and user interaction.

---

## 🎯 Features

- 👁️ Real-time facial landmark detection  
- 😴 Fatigue & drowsiness detection (EAR, MAR, PERCLOS)  
- ⚡ Adaptive HMI response based on driver state  
- 🌙 Works in varying lighting conditions  
- 🔄 Scalable to ADAS integration  

---

## ⚙️ How It Works

Camera Input → Face Detection → Landmark Detection  
→ Feature Extraction (EAR, MAR, PERCLOS)  
→ Driver State Classification → Alert/HMI Response  

---

## 📊 Validation

- Tested on 300+ users  
- Real-time performance (~25–30 FPS)  
- Accuracy ~90%  

---

## 🔧 Tech Stack

- Python / OpenCV
