Project Overview:

HCAI Monitoring System is a Human-Centered AI-based solution designed for real-time driver state monitoring using facial behavior analysis. The system leverages computer vision techniques to continuously analyze facial features and detect critical driver conditions such as fatigue, drowsiness, and reduced attention levels. By extracting geometric features like Eye Aspect Ratio (EAR), Mouth Aspect Ratio (MAR), and temporal metrics such as PERCLOS, the system ensures accurate and reliable detection of driver states. Unlike traditional approaches that rely solely on environmental sensing, this system focuses on the human factor, making it a key component for next-generation intelligent mobility solutions. The architecture is designed to operate in real time, providing low-latency feedback and ensuring immediate response to potentially dangerous situations. A core feature of the system is its adaptive Human-Machine Interface (HMI), which dynamically adjusts alerts and interactions based on the driver’s condition. This enables a more intuitive and context-aware user experience, reducing unnecessary distractions while enhancing safety. The system is scalable and designed to integrate seamlessly with Advanced Driver Assistance Systems (ADAS), allowing driver-state insights to contribute to broader vehicle decision-making. For validation purposes, this website was created to test the system across diverse real-world user scenarios and interactions. Overall, the HCAI Monitoring System represents a step toward safer, smarter, and more human-aware automotive technologies.
