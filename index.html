Project Overview:

HCAI (Human Centered Artificial Intelligence) is a real-time driver monitoring system designed to analyze facial behavior using computer vision and parametric state modeling. The system measures Eye Aspect Ratio (EAR), Mouth Aspect Ratio (MAR), blink rate, head orientation, and sustained eye closure to classify driver states into Happy, Neutral, Stress, Anger, and Fatigue categories. It applies temporal smoothing and stabilization logic to prevent rapid state switching, ensuring reliable and automotive-grade behavioral assessment. Fatigue detection is prioritized as a safety-critical state and triggers alerts when sustained low EAR or abnormal blink patterns are detected. The system runs fully in-browser using TensorFlow.js and MediaPipe FaceMesh, making it privacy-friendly and deployable without backend infrastructure. It demonstrates human-centered AI principles, explainable rule-based classification, and safety-focused monitoring suitable for next-generation intelligent cockpit systems.
